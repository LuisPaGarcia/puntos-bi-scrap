# Puntos Bi Scrap

This project uses a configuration file `config.env` to store environment variables. This needs Node v20.6.0.

## Configuration

The `config.env` file contains the following variables:

- `NUMERO_DPI`: This is the CUI number without spaces.
- `FECHA_NACIMIENTO`: This is the birth date in the format DD/MM/YYYY.
- `NUMERO_TARJETA_CLUB_BI`: This is the club BI card number.

Please ensure to update these variables with your own values before running the project.

## Running the Project

To run the project, follow these steps:

1. Update the `config.env` file with your own values.
2. Run the command `npm install`.
3. Run the command `npm run start`.
